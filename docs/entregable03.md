## Descripción del proyecto

En este repositorio documentaré mi avance por el curso en línea Laravel From Scratch 2026 de Laracasts, siguiendo los episodios asignados como parte del Proyecto 1 del curso ISW-811 Aplicaciones Web Usando Software Libre, impartido por el profe Misael Matamoros Soto.
